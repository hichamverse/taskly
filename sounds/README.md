# Sound Files Directory

Place your sound files in this directory:

## Required Sound Files

- `completed.mp3` - Sound played when a task is completed or habit is marked as done
- `reminder.mp3` - Sound played for habit reminders

## File Requirements

- Format: MP3 (recommended for browser compatibility)
- Keep files small for fast loading
- Use short, pleasant notification sounds

## Example Sound Sources

You can find free notification sounds from:
- Freesound.org
- ZapSplat.com
- Notification Sounds (mobile apps)

## Usage

The sounds will be automatically loaded by the application when:
- Tasks are completed
- Habits are marked as done
- Habit reminders trigger
- Counter habits are incremented

Sounds can be toggled on/off in Settings.
